## Context

El proyecto "web" es un scraper de propiedades que usa Playwright para obtener HTML renderizado de páginas como metrocuadrado.com. El README actual solo tiene una línea "# modulo de prueba" sin ninguna documentación útil.

## Goals / Non-Goals

**Goals:**
- Crear un README.md completo y profesional
- Incluir todas las secciones estándar de un proyecto bien documentado
- Mantener el documento conciso pero informativo

**Non-Goals:**
- No modificar código existente
- No agregar nueva funcionalidad al proyecto

## Decisions

- Usar formato markdown estándar para README
- Incluir: descripción, instalación, uso, estructura, pruebas, contribución
- Mantener tono profesional pero accesible

## Risks / Trade-offs

- Ninguno. Solo es documentación.
